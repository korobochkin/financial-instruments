== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Financial Instruments'
3. Activate Financial Instruments from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.

= From WordPress.org =

1. Download Financial Instruments.
2. Upload the 'Financial Instruments' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...).
3. Activate Financial Instruments from your Plugins page.
4. Add widgets on yourdomain.com/wp-admin/widgets.php page.
